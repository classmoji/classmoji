---
title: Contributing
description: You can contribute to Classmoji in many ways.
---

Go to our [GitHub repository](https://github.com/classmoji/classmoji) and open an issue or a pull request. We are always looking for contributors to help us improve Classmoji.

We highly encourage students and developers at all levels to contribute. First-time contributors are always welcome!

You can contribute in many ways, including:

- Reporting bugs
- Suggesting new features
- Writing documentation
- Writing code
- Reviewing code
- Sharing the app with others
- Giving feedback
- And more!
