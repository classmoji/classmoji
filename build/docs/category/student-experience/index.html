Student Experience

📄️ 🧑‍💻 Account Setup

Getting started with Classmoji takes just a few steps—and you only need to do it once.

📄️ 📤 Submissions

In Classmoji, assignments are submitted by closing the GitHub issue assigned to you.

📄️ 🪙 Using Tokens

Tokens are a flexible way to manage deadlines without needing to email your instructor. You can spend tokens to submit late, request extensions, or (in some classes) earn extra credit.
